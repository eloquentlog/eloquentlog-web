## Repository

https://gitlab.com/eloquentlog/eloquentlog-web-frontend


## Requirements

* Node.js


## Setup

```zsh
% npm install
```

## Build

```zsh
% make build
```


## Development

### Vet

```zsh
: using `tslint`
% make lint
```

### Run

```zsh
% make serve
```
